# Vue-Custom-Cursor

Include the component in `App.vue`.

## Settings

| Option      | Type         |
| ------------- |:-------------:|
| targets     | Array | 


```
    <custom-cursor
      :targets="['img', 'a', 'button', 'your-hover-class']"
      :circleColor="'#fff'"
      :circleColorHover="'#2f2f2f'"
      :dotColor="'#2f2f2f'"
      :dotColorHover="'lightgray'"
      :hoverSize="1.8"
    ></custom-cursor>
```
